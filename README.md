## What this site is

This project is a static external guide built for Cloudflare Pages. It helps visitors move from product cover images to category pages, detail pages, quality-control checks and long-form product research content.

## What this site is not

Hacoo VIP is not the official Hacoo app website. It does not manufacture, sell, store, ship or process payment for products. It is an independent navigation and research layer.
