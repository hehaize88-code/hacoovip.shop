ACBuy Return and Exchange Guide: Warehouse Deadlines, Fees and Refunds

How the current ACBuy return interface works, why the warehouse clock matters, and what shoppers should document before asking an agent to contact the seller.

Fact-checked July 2026 · 10–12 minute read · Independent guide

A return through a shopping agent is not the same as taking an item back to a local shop. The product was bought from a third-party seller, delivered to ACBuy’s warehouse and inspected through photographs rather than in your hands. If something is wrong, the request has to move back through the purchasing agent to the seller before the item leaves China. That extra layer makes timing and evidence more important than a long explanation written several days later.

ACBuy’s current order interface includes separate return and exchange options, a list of application reasons, a warehouse eligibility message and fee information. Those details create a practical framework, but they are not a promise that every request will be approved. Seller rules, product type, the stated reason, warehouse timing and responsibility for the problem can all affect the result.

Start with the warehouse deadline

The clearest limit in ACBuy’s current English interface says that a product stored for more than five days cannot be returned or exchanged. Treat that as an action deadline, not as five days to begin thinking. Warehouse processing, photo review and communication all take time, so inspect an item as soon as it becomes available.

The five-day message concerns the domestic after-sales stage shown in the platform. It should not be confused with ACBuy’s general storage period or with an overseas consumer return right. Once the item is placed into an international parcel, the simple warehouse-to-seller route is no longer the same process. Do not submit an item for international delivery while a return question is unresolved.

Open the order record, save the arrival date and review the QC images on the same day if possible. If an image is unclear, use the account tools to request clarification without waiting for the end of the eligibility window.

Return or exchange: decide what outcome you actually need

A return sends the item back and seeks a refund through the seller’s process. An exchange asks the seller to replace it, which creates another domestic shipment and another warehouse check. ACBuy’s interface currently describes a return as usually taking 7–15 days and an exchange as usually taking 7–20 days. These are interface estimates, not guaranteed completion dates.

An exchange can make sense when the seller has the correct size or colour and the item is otherwise suitable. A return may be cleaner when the listing was misleading, the correct version is unavailable or the replacement would repeat the same uncertainty. Before choosing, check whether the seller still offers the desired variant and whether additional domestic shipping may apply.

Use the correct application reason

The current ACBuy return form presents reasons that include buying the wrong item or no longer liking it, receiving the wrong product from the seller, a product-quality issue, failure to ship within the agreed time and an “other” option. These categories matter because they help determine who may be responsible for fees.

Choose the reason that matches the evidence. Do not turn a change of mind into a quality complaint, and do not select “wrong product” until you compare the warehouse item with the exact variant you authorized. A manual or DIY order is especially dependent on the size, colour, model and remarks you submitted.

Keep the explanation short and testable. “Ordered navy, warehouse label and item are black” is more useful than “this looks bad.” If the issue is a measurement, show the requested measurement and the visible result. If it is damage, identify the image and location instead of asking the buyer to search every photograph.

Evidence to collect before applying
  • The live or saved source listing and selected variant.
  • The ACBuy order number and warehouse arrival date.
  • QC images showing the relevant label, colour, damage or missing part.
  • A concise comparison between what was ordered and what arrived.
  • The preferred outcome: return, exchange or clarification.
  • Any seller message or condition that affects after-sales eligibility.

QC photos support a claim, but they have limits

Warehouse photographs can document visible condition, quantity, colour and labels. They cannot reliably prove authenticity, fibre composition, internal electronics or long-term durability. Lighting and compression can also change colour and texture. A return request is stronger when it points to an observable difference rather than a conclusion the image cannot establish.

ACBuy’s interface warns that minor quality issues such as glue marks, flaws or stains may not be accepted by the seller as a reason for the seller to pay return shipping. That does not mean the shopper cannot apply; it means responsibility for the domestic return cost may remain disputed. Review the warehouse QC checklist before deciding whether a photographed detail is clear enough to support the requested outcome.

Understand the current service fee and shipping cost

As displayed in ACBuy’s English return interface on 15 July 2026, the first return or exchange application each month is described as free of the operation service charge. The same interface lists a return and exchange service fee of CNY 5 per order outside that free operation entitlement. Because account benefits and platform terms can change, verify the amount shown in your own application before paying.

The operation fee is separate from domestic return or exchange shipping. ACBuy states that shipping generated by the return or exchange is negotiated between the agent and seller and collected by ACBuy. The interface also says that even when an exchange is connected to a seller reason, the shopper may still have to bear part of the exchange shipping cost.

Responsibility matters again after review. The current wording says that if the return or exchange is ultimately the responsibility of the purchasing agent, seller or platform, the CNY 5 operation fee is refunded to the shopper’s ACBuy balance. Do not assume that every initial payment will be refunded; wait for the result of the responsibility review shown for the order.

What happens after a refund request

ACBuy’s current interface explains that after a refund application is submitted, the buyer checks the product’s status. It also warns that the application cannot be cancelled before that verification is complete. This is another reason to choose the correct item and outcome before pressing submit.

During review, watch for messages asking for more detail or confirming a seller response. The seller may accept, reject or propose another solution. If an exchange is approved, the replacement still needs to return to the warehouse and be checked against the original order. Do not treat an exchange approval as proof that the next item will automatically be correct.

A refund can also involve several stages: approval, domestic return, seller receipt and the money moving through the platform process. The interface’s “usually” time ranges should be used for planning, not as a deadline that guarantees funds on a specific date.

Common return mistakes that reduce your options

Waiting until parcel submission

Combining items quickly may feel efficient, but it removes inspection time. Keep newly stored items out of a parcel until the visible details and variant match are confirmed.

Using an old screenshot as the entire claim

A screenshot can show what you saw, but it may omit the selected variant or seller conditions. Pair it with the order record, source link and warehouse photographs.

Ignoring your own selection

If the received item matches the option you approved, the case is not the same as a seller sending the wrong item. Check the cart and order specification before assigning responsibility.

Assuming a visible imperfection makes return shipping free

The seller may dispute responsibility for minor marks. Separate eligibility to request a return from the question of who pays the domestic shipping.

A practical after-sales sequence

  1. Open the warehouse item immediately and note the storage date.
  2. Compare the item with the paid order, selected specification and source listing.
  3. Identify the exact visible mismatch and save supporting images.
  4. Choose return or exchange based on stock, cost and the desired result.
  5. Select the accurate reason and write a short factual explanation.
  6. Review the live operation fee and any domestic shipping request.
  7. Submit once, then monitor the buyer’s verification and seller response.
  8. For an exchange, inspect the replacement before international shipping.

This sequence fits inside the broader ACBuy ordering workflow: product purchase, domestic delivery, warehouse review and only then international parcel creation. Keeping those stages separate is the easiest way to preserve after-sales options.

The sensible rule: inspect early and argue narrowly

A strong return application does not need dramatic language. It needs a timely request, the correct reason and a visible comparison between the authorized order and the warehouse result. When the issue is subjective or too small for a photograph to establish, plan for the possibility that the seller will not cover domestic return shipping.

Most avoidable losses happen before the return form: an unclear variant was approved, the source page was not saved, QC images were ignored or the five-day warehouse window passed. Fixing that routine is more valuable than trying to find a universal “free return” promise that the current ACBuy interface does not make.
