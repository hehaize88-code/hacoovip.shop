# AllChinaBuy Pro

Independent China shopping directory and editorial research site for
`allchinabuy.pro`.

## Included

- responsive landing page and product discovery directory
- category routes into live AllChinaBuy search results
- product detail pages, buying guides, FAQ and policy pages
- English, French, German, Italian and Spanish landing routes
- sitemap, robots, manifest and structured data

## Development

Requires Node.js 22 or newer.

```bash
npm ci
npm run dev
```

Production validation:

```bash
npm run lint
npm run build
```

The hosted inspection version is available at
<https://allchinabuy-pro.hehaize88.chatgpt.site>.
